Συνάρτηση που την υπολογίζει στην Pascal

function f(x, y, n: integer) : integer;
begin
	if n = 0 then
		f := y + 1;
	else 
		if y = 0 then
			case n of
				1 : f := x;
				2 : f := 0;
				3 : f := 1;
			else f := x;
			end;
		else
			f := f(x, f(x, y-1, n), n-1);
end;

Στην loop αυτή η συνάρτηση δεν μπορεί να υπολογιστεί γιατί δεν είναι πρωταρχικά αναδρομική.
